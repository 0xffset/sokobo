\subsubsection*{template$<$typename T$>$\newline
class Matrix$<$ T $>$}
A templated matrix class supporting various linear algebra operations. 

This class provides a comprehensive implementation of matrix operations including basic arithmetic, advanced operations (determinant, inverse, eigenvalues), matrix decompositions (LU, QR), and iterative algorithms (Lanczos). Supports both real arithmetic types and complex number types.

T \doxymbox{\hyperlink{class_matrix}{Matrix}}$<$ T $>$::\+determinant (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption}) const}



Computes the determinant of the matrix. 

The calculation follows these rules based on the matrix dimension  $ n
\times n $:\+


\begin{DoxyItemize}
\item For $ n = 1 $:\+ \[ \det(A) = a_{1,1} \]
\item For $ n = 2 $:\+ \[ \det(A) = a_{0,0}a_{1,1} - a_{0,1}a_{1,0} \]
\item For $ n > 2 $, using {\bfseries{LU Decomposition}} $ A = LU $:\+ \[ \det(A) = \det(L)\det(U) = \prod_{i=0}^{n-1} u_{i,i} \] \texorpdfstring{$\ast$}{*}(Note:\+ $ \det(L) = 1 $ as it is a unit triangular matrix)\texorpdfstring{$\ast$}{*}.
\end{DoxyItemize}
